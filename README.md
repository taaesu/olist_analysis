\## 프로젝트 개요

\- 브라질 전자상거래 플랫폼 Olist의 공개 데이터셋을 활용하여, 고객 만족도를 저해하는 주요 요인을 파악하고, 특히 리뷰 점수에 부정적인 영향을 미치는 요인을 규명하는 것을 목표로 분석을 진행했습니다.

\- 실제 데이터 분석가의 업무 프로세스를 가정하여, SQL과 Python(pandas, seaborn, scipy 등)을 이용한 EDA(탐색적 데이터 분석) → 가설 설정 및 검정 → 인사이트 도출의 흐름으로 프로젝트를 설계했습니다.



\## 데이터셋

\- 출처: Kaggle – Brazilian E-Commerce Public Dataset by Olist

\- 주요 테이블:

* orders: 주문 ID, 구매/승인/배송 관련 날짜
* order\_items: 주문별 상품, 배송비, 판매자 정보
* customers: 고객 ID, 도시(city), 주(state)
* products: 상품 ID, 카테고리
* sellers: 판매자 정보
* order\_payments: 결제 금액, 결제 횟수
* order\_reviews: 리뷰 점수, 리뷰 텍스트
* geolocation: 위도/경도, 도시, 주(state) 정보
* product\_category\_name\_translation: 상품 카테고리명(포르투갈어 → 영어 번역 매핑)



\## 분석 과정



1\. 문제 정의

\- 비즈니스 질문: “리뷰 점수가 낮아지는 주요 원인은 무엇인가?”



2\. 데이터 준비

\- SQLite DB에서 SQL로 raw 데이터 추출

\- pandas로 결측치 및 이상치 처리

\- 배송 지연 여부(delay\_days > 0) 계산하여 파생 변수 생성



3\. 탐색적 분석 (EDA)

\- 리뷰 점수 분포 확인 -> 평균이 전반적으로 높은 편이지만 일부 낮은 점수 집중

\- 배송 지연과 리뷰 점수 간 관계 확인



4\. 가설 검정

\- 가설: 배송 지연이 리뷰 점수에 부정적 영향을 준다

\- 독립표본 t-검정 수행

\- 결과: p-value ≈ 0 -> 유의하게 차이 존재

\- 배송 지연 시 리뷰 점수가 더 낮다는 점을 통계적으로 확인



\## 주요 결과

\- 배송 지연은 리뷰 점수를 유의하게 낮춘다

\- 배송비, 결제 방식 등은 리뷰 점수에 상대적으로 약한 영향



\## 인사이트
